# Security Policy

## Scope

speedkit.eu is a **static HTML index**. It has no:
- Server-side code
- User input processing
- Authentication
- Data storage
- API endpoints

## Reporting

For factual corrections to index content, open a GitHub issue.

There is no security vulnerability surface. This is a static file.

## Non-applicable

The following are out of scope:
- XSS (no JavaScript execution context)
- CSRF (no forms or state)
- Injection (no input processing)
- Authentication bypass (no authentication)

## Contact

No private security contact exists. All changes are public via GitHub.
