# Maintenance Mode

**Status: MAINTENANCE ONLY**

## Accepted changes

- New system additions (append-only)
- Status updates for existing systems
- Checksum updates
- Signed snapshots

## Rejected changes

- Deletions
- Reordering
- HTML reformatting
- Styling changes

## Change process

1. Open issue describing addition
2. Add new entry (append-only)
3. Update INDEX_CHECKSUM.txt
4. Create signed snapshot
