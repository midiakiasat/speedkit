# Tagging Strategy

Initial tag: v1.0.0

Tags indicate structural state, not features. No semantic feature versioning.

New tags are created only for:

- Append-only system addition (v1.1.0, v1.2.0, etc.)
- Formal decommission notice (v1.0.1, v1.1.1, etc.)

Tags reflect repository structure changes, not content modifications.
